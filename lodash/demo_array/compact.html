过滤数组

  • _.compact(array)

    过滤 false, 保留其他值

  • _.difference(array, [values])
    _.differenceBy(array, [values], function(val) { return '比较结果'，比较结果不同的保留})
    _.differenceWith(array, [values], function(arrVal, othVal) { return false 的保留 })

    在 array 中，不在 values 中

  • _.drop(array, [n=1])：排除前面 n 个元素
    _.dropRight(array, [n=1])：排除后面 n 个元素
    _.dropRightWhile(array, function(value, index, array) { return false 时作为起点 })
    _.dropWhile(array, function(value, index, array) { return false 时作为终点 })
  • _.initial(array)：排除数组最后一个元素
  • _.intersection(arrays)
    _.intersectionBy(arrays, function(val) { return 比较结果 })
    _.intersectionWith(arrays, function(arrVal, othVal) { return true 保留 })

    在 array 中，同时也在 values 中（数组交集）
