# 宽度

定义元素在不同视口大小中的宽度。


## 用法

```
<div class="uk-width-*">
  添加一个 .uk-width-* class 到元素来确定它的大小。
</div>

<div class="uk-width-1-1">填充可用宽度的 100%</div>
<div class="uk-width-1-2">元素占用其父容器的一半</div>
<div class="uk-width-1-3">元素占用其父容器的三分之一</div>
<div class="uk-width-1-4">元素占用其父容器的四分之一</div>
<div class="uk-width-1-5">元素占用其父容器的五分之一</div>
<div class="uk-width-1-6">元素占用其父容器的六分之一</div>
```


## 自动宽度 & 扩展宽度

```
<div class="uk-width-auto">
  扩展到其自身内容的宽度
</div>

<div class="uk-width-expand">
  扩展以填充容器的所有剩余空间
</div>
```

## 等分网格

```
<div class="uk-child-width-1-2">所有子元素占用其父容器的一半</div>
<div class="uk-child-width-1-3">所有子元素占用其父容器的三分之一</div>
<div class="uk-child-width-1-4">所有子元素占用其父容器的四分之一</div>
<div class="uk-child-width-1-5">所有子元素占用其父容器的五分之一</div>
<div class="uk-child-width-1-6">所有子元素占用其父容器的六分之一</div>
<div class="uk-child-width-auto">根据内容的尺寸大小将网格划分为相等的单位</div>
<div class="uk-child-width-expand">根据可用宽度空间将网格划分为相等的单位</div>
```

## 固定宽度

```
<div class="uk-width-small">应用固定宽度 150px</div>
<div class="uk-width-medium">应用固定宽度 300px</div>
<div class="uk-width-large">应用固定宽度 450px</div>
<div class="uk-width-xlarge">应用固定宽度 600px</div>
<div class="uk-width-xxlarge">应用固定宽度 750px</div>
```

## 混合宽度

```
<div class="uk-child-width-expand">
    <div>
      1
    </div>
    <div class="uk-width-1-3">
      2
    </div>
    <div>
      3
    </div>
    <div>
      4
    </div>
</div>
```
